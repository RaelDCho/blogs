# Blog App
---
Blog app that allows users to login, create blogs, like blogs, and delete blogs that they have been created.


### How To Run:
---
`npm install`

`npm run start` (in both client and server directories)


### How It's Made:
---
**Tech Used:** HTML, CSS, JavaScript, React, Node.js, Express, MongoDB (Will be replaced for SQL)
